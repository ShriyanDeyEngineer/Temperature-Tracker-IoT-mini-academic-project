# Temperature-Tracker-IoT-mini-academic-project
IoT temperature tracking system using Particle Photon 2 and C++ with real time sensor data integrated into Google Sheets through cloud scripting. HTML/JavaScript web interface for fetching and displaying live sensor readings using API requests and dynamic updates.

This project was started and completed during April, 2026.
